## Installation
1. Go to Spring Initializr and make sure to pick all the dependencies before generating the project: https://start.spring.io/
2. NPM
```sh
npm install npm@latest -g
 ```
3. Tailwind CSS
```bash
npm install -D tailwindcss
```
